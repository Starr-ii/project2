About this site

This from main website is a demonstration site for the tutorial series "How To BUILD UP" on DigitalOcean's community platform. Learn how to recreate and cutomize this site with HTML by following the following step-by-step instructions in the tutorial series. Participants will finish with an understanding of HTML and a website ready to deploy to the cloud. No coding experience necessary!
